## ðŸ“Š **Portfolio Management Prompts Explained**

### **#12 - Portfolio Analysis (Basic)**
Creates or analyzes a portfolio with specified allocations. Returns summary statistics, holdings, and key metrics.

### **#13 - Portfolio Holdings**
Shows all holdings in your portfolio with weights, shares, prices, market values, and fundamental metrics.

### **#14 - Portfolio Exposure**
Analyzes sector exposure, factor exposure (beta, momentum, value), and concentration metrics.

### **#15 - Portfolio Optimization**
Optimizes portfolio allocation for maximum Sharpe ratio or other objectives. Shows current vs. optimized weights and expected performance.

### **#16 - Portfolio Risk Analysis (CVaR)**
Calculates Conditional Value at Risk (CVaR) - expected loss in worst-case scenarios. More conservative than VaR.

### **#17 - Portfolio Performance Attribution**
Shows what's driving portfolio performance using Brinson-Fachler attribution (allocation, selection, interaction effects).

### **#18 - Portfolio Stress Testing**
Tests portfolio performance under different market scenarios (e.g., market crash, sector decline).

### **#19 - Portfolio Diversification**
Analyzes diversification using HHI, effective number of holdings, and risk contribution metrics.

### **#20 - Portfolio Export**
Exports portfolio analysis as PowerPoint, PDF, or Excel for presentations and reports.
